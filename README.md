# chef-wrapper-marathon-cookbook

Wrapper cookbook to install Marathon.

## Supported Platforms

Ubuntu 14.04

## Usage

### chef-wrapper-marathon::default

Include `chef-wrapper-marathon` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[chef-wrapper-marathon::default]"
  ]
}
```
